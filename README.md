# Car_Rental_Management

Desktop-based Car Rental Management System developed using **Java Swing**, **MySQL**, and the **Object-Oriented Programming (OOP)** approach. This application was created as the final project for the **Object-Oriented Programming II** course.

## Description

Car Rental Management is a desktop application designed to simplify and organize car rental business operations. The system enables administrators to manage vehicle data, customer data, rental transactions, vehicle returns, and generate reports efficiently.

The application implements the four main principles of Object-Oriented Programming (OOP): **Encapsulation**, **Inheritance**, **Polymorphism**, and **Abstraction**, resulting in a modular, maintainable, and scalable application.

---

## Features

* User Authentication (Login & Logout)
* Vehicle Management (CRUD + Soft Delete)
* Customer Management (CRUD + Soft Delete)
* Rental Transaction Management
* Vehicle Return Management
* Automatic Rental Cost Calculation
* Automatic Late Return Fine Calculation
* Report Generation using JasperReport
* Search Functionality
* Real-time Vehicle Status Management
